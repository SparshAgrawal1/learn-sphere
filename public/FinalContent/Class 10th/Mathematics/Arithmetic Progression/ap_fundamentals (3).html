What is an Arithmetic Progression?
Basic Definition
An Arithmetic Progression (AP) is a sequence of numbers where each term after the first is obtained by adding a fixed constant to the previous term.
Example: 2, 5, 8, 11, 14... (adding 3 each time)
General Form of AP
Standard Representation
Every arithmetic progression can be written in this general form:
a, a+d, a+2d, a+3d, a+4d, ...
where a = first term and d = common difference
